# Box2D CarRacing PPO

Implementation of Proximal Policy Optimization(PPO) algorithm for the
[`CarRacing-v2`](https://gymnasium.farama.org/environments/box2d/car_racing/)
Box2D environment from OpenAI's Gym library.
